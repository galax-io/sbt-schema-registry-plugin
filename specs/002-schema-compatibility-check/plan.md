## Summary

Add a `schemaRegistryTestCompatibility` sbt task that checks local schema files against the registry's compatibility rules before registration. Uses `SchemaRegistryClient.testCompatibilityVerbose` to return specific incompatibility messages. Reuses existing `schemaRegistryRegistrations` setting — zero new configuration. Pure core (`CompatibilityChecker`) with effects at the sbt task edge.
